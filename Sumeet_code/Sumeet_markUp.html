Design System & Style Guide

Sumeet Vishwakarma Portfolio - Version 1.0

Introduction

This comprehensive style guide ensures consistency and quality across the portfolio website. It serves as a single source of truth for design decisions and implementation guidelines.

0. Brand Identity

Voice & Tone

  • Professional and confident
  • Technical yet approachable
  • Clear and concise

Brand Values

  • Technical Excellence
  • Innovation
  • User-Centric Design

1. Color System

Color Usage Guidelines

  • Primary: Headers, navigation
  • Secondary: CTAs, links
  • Text: Content hierarchy

Accessibility

All color combinations meet WCAG 2.1 AA standards

7. Accessibility Guidelines

Color Contrast

  • Text: 4.5:1 minimum contrast ratio
  • Large Text: 3:1 minimum contrast ratio

Keyboard Navigation

  • Visible focus states
  • Logical tab order
  • Skip navigation link

Screen Readers

  • ARIA labels where needed
  • Semantic HTML structure
  • Alternative text for images

8. Implementation Guidelines

CSS Architecture

styles/ ├── base/ ├── components/ ├── layouts/ └── utilities/

Performance Guidelines

  • Critical CSS inlined
  • Lazy loading for images
  • Minified and compressed assets
